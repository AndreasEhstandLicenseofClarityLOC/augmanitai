DESIGN DECISIONS
================

1. Mock LLMs: All examples use mock LLMs for offline execution
   - No API keys required
   - Deterministic behavior for testing
   - Can be swapped for real LLMs (Claude, GPT-4, etc.)

2. Shared AUGMANITAI Pattern: All examples integrate AugmanitaiDB.search()
   - Shows consistent API usage across frameworks
   - Demonstrates database queries in agent context
   - Uses ~25 terms from AUGMANITAI database

3. Progressive Complexity:
   - LangGraph: Simple linear StateGraph (easiest to learn)
   - CrewAI: Role-based agents with structured tasks (medium complexity)
   - AutoGen: Async group chat with round-robin conversation (most advanced)

4. Test Coverage:
   - LangGraph: 4 tests (query parsing, retrieval, composition, E2E)
   - CrewAI: 4 tests (output, terminology, relevance, multi-question)
   - AutoGen: 5 tests (response, addressing, participation, variety, structure)

AUGMANITAI Integration Points
==============================

All examples use these core AUGMANITAI methods:

1. AugmanitaiDB.search(query, domain=None, threshold=0.3, limit=None)
   - Fuzzy string matching on term definitions
   - Returns (Term, score) tuples
   - Used by: term_retriever (LangGraph), terminology_agent (CrewAI), search_terminology (AutoGen)

2. AugmanitaiDB.get_domain(domain)
   - Returns all terms in a domain
   - Used for domain-specific lookups
   - Domains: cognitive_processes, linguistic_dynamics, systemic_effects, etc.

3. AugmanitaiDB.get_term(term_id)
   - Direct term lookup by ID
   - Returns full Term object
   - Used when term ID is known

4. AugmanitaiDB.get_related_terms(term_id, direction="both")
   - Finds connected terms
   - Used for terminology context and relationships

EXAMPLE WORKFLOW
================

User Query: "How do AI systems influence human language adoption and terminology usage?"

LangGraph:
  1. query_parser → extracts ["SLE", "RKE", "Compound Cognition"]
  2. term_retriever → db.search() → retrieves definitions
  3. response_composer → formats structured answer

CrewAI:
  1. Research Agent → analyzes question into concepts
  2. Terminology Agent → searches AUGMANITAI for each concept
  3. Writer Agent → synthesizes into 300-500 word response

AutoGen:
  1. Query Analyzer → identifies key concepts
  2. Terminology Expert → looks up definitions
  3. Synthesis Officer → composes response
  (Round-robin conversation, agents see each other's work)
