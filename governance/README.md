## New to the project?

1. Read the top-level project `README.md` to understand what AUGMANITAI is.
2. Read `GOVERNANCE.md` to understand how decisions are made.
3. Read `CONTRIBUTING.md` to understand how to make your first contribution.
4. Read `CODE_OF_CONDUCT.md` before engaging with the community.

## Propose a new term?

Open a Term Proposal using the template at `.github/ISSUE_TEMPLATE/term_proposal.yml`. Every field is required. Proposals missing ISO 704 characteristics, attestation sources, or differentia will be returned for revision.

## Report a security issue?

Do NOT open a public issue. See `SECURITY.md` for the private reporting channel.

## Questions about licensing?

See the `../licensing/` directory for the full license texts and the dual-license explanation.
